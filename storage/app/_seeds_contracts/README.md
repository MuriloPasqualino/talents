# Modelos de contrato (seed)

Coloque aqui os 3 arquivos `.docx` antes do deploy (nomes esperados pelo `ContractTemplateSeeder`):

- `TALENTS - CONSULTORIA - NOVO.docx`
- `TALENTS - CONTRATACAO DE TALENTOS - NOVO.docx` ou `TALENTS - CONTRATAÇAO DE TALENTOS - NOVO.docx`
- `TALENTS - PALESTRA - NOVO.docx`

Sem esses arquivos o seeder apenas registra aviso no log e não falha.
